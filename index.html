O que é HTML?

topo

HTML (Linguagem de marcação de hipertexto) é uma linguagem usada para estruturar a maioria das páginas da internet. Através de tags, o html oferece a estrutura básica para criação de conteúdos na web. Não é uma linguagem de programação, mas sim de marcação, pois usa as tags mancionadas antes para maracar as partes do conteúdo e definir como devem ser exibidas no navegador.

Estrutura básica do HTML

topo

A estrutura básica se refere as três tags básicas do HTML.
A primeira tag é a html. Ela diz ao navegador que tudo que esta dentro de seu corpo é um arquivo html. É a tag que contém também as outras duas principais tags que que compõe a estrutura básica ddo html.

A tag head são tags que contém dados que não serão exibidos no corpo do site. O conteúdo do head, apesar de não ser exibido, é de grande importância, pois é nessta seção que definios o título da página, links para as folhas de estilo e scripts e metadados para motores de busca.

A última tag da estrutura báscica é o body. No body definimos tudo que será exibido para o usuário que navega pelo site. Todas as imagens, textos e links são inseridos no corpo desta tag.

Tags html aprendidas

Tags h1 a h6

topo

O elemento h1 a h6 são usados para definir títulos e subtítulos no HTML. O uso correto auxilia os mecanismos de buscas a compreender a estrutura do texto do site.

Elemento p

topo

Basicamente, o elemento p é usado para definir um parágrafo.

Elemeto mark

topo

Dá destaque a parte do texto envolvida nesta tag. Deixa o texto com a aparência de um marca texto.

Elemento small

topo

Este elemento é usado para indcar informações secundárias. Ele deixa o texto com aparência de letras miúdas.

Elemento i

topo

Destaca o texto dando a ele a aparência de itálico. Normalmente usado para destacar textos em outros idiomas termos técnicos e expressões de personagens.

Elemento u

topo

O texto envolvido neste elemento é exibido com um sublinhado para o usuário.

Elemento strong

topo

Este elemento é usado para dar ênfase ao texto, indicando a grande importância que possui. O texto é apresentado em negrito.

Elemento ol

topo

Este elemento é usado para criar uma lista de itens ordenados onde a ordem dos elementos é relevante

Elemento ul

topo

É usado para criar uma lista de itens onde a ordem dos itens não tem relevância.

Elemento li

topo

É usado dentro dos elemento ol e ul para definir os itens que compõe a lista.

Elemento a

topo

O elemento a (âncora), é utilizado para criar hiperlinks.

Elemento hr

topo

Este elemento cria uma linha horizontal no texto. Antes era apenas visual, mas atualmente tem valor semântico, indicando uma mudança de tema no texto

Elemento sub

topo

Elemento usado para criar efeito subscrito no texto. As vezes também exibe o texto em fonte menor. Útil para fórmulas químicas, matemáticas e notas de rodapé.

Elemento sup

topo

Este elemento é utilizado para gerar um texto sobrescrito. O texto envolto nesta tag fica meia linha acima do texto. É usado para referências bibliográficas, marcadores e notas de rodapé.

Elemento blockquote

topo

O elemento blockquote é ideal para destacar citações longas no conteúdo do texto, como trechos de livros, artigos ou discursos.

Elemento font

topo

Era usado para definir a fonte do texto, tamanho e cor. No HTML 5 seu uso não é recomendado, sendo melhor o uso de CSS para fazer essas edições.

Elemento del

topo

Del é o elemento de texto excluído. O texto envolvido nesta tag fica com uma linha sobre o texto. Indica que o texto foi excluído ou alterado.

Elemento abbr

topo

Este elemento representa uma abreviação no texto.
